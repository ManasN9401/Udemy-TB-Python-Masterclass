# Python Masterclass - Udemy

This repository contains code examples and exercises from **Tim Buchalka's Python Masterclass** course on Udemy.

## Course Overview

A comprehensive Python programming course covering fundamentals through advanced topics including GUI development with Tkinter, threading, and practical projects.

## Repository Structure

```
.
├── FlowControl/          # Conditionals, loops, and program flow
├── Sequences/            # Lists, tuples, and data structures
├── Threading/            # Multi-threading concepts
├── Tkinter/              # GUI development with Tkinter
└── *.py                  # Root-level exercise files
```

## Topics Covered

### Basics
- Hello World and basic output
- String operations and formatting
- Operators and expressions
- Escape characters and special characters
- Range functions

### Flow Control (`FlowControl/`)
- Conditions and boolean logic
- If/else statements
- For loops and while loops
- Input validation
- Interactive text adventures
- Shopping cart examples

### Sequences (`Sequences/`)
- Lists (mutable vs immutable)
- Tuples
- List comprehensions
- Sorting and filtering
- Nested data structures
- Unpacking and enumeration
- Practical projects (games, battle ships)

### Advanced Topics
- **Threading**: Multi-threaded programming concepts
- **Tkinter**: Building GUI applications with Python's standard GUI library

## Getting Started

### Prerequisites
- Python 3.x installed
- No additional dependencies required (uses standard library)

### Running the Code
```bash
# Run any Python file directly
python HelloWorld.py
python FlowControl/adventure.py
python Sequences/simplegame.py
python Tkinter/tkinterdemo.py
```

## File Highlights

| File | Description |
